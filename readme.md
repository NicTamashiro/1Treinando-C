# Beecrowd — Soluções em C
 
Repositório com soluções dos problemas resolvidos na plataforma [Beecrowd](https://www.beecrowd.com.br/), uma das maiores plataformas de juiz online do Brasil voltada para prática de programação e preparação para competições.
 
## Sobre
 
Os problemas estão organizados por categoria, seguindo a estrutura da própria plataforma. Cada arquivo é nomeado com o ID do problema correspondente.
 
O foco principal está nos fundamentos da programação — lógica, manipulação de dados e resolução de problemas — utilizando a linguagem C.
 
## Tecnologias
 
| Ferramenta | Descrição |
|------------|-----------|
| C | Linguagem principal utilizada nas soluções |
| GCC | Compilador C |
| Beecrowd | Plataforma de submissão e validação |
